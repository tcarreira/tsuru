=======================
tsr 0.3.6 release notes
=======================

Welcome to tsr 0.3.6!

These release notes cover the `new features`_ and `backwards incompatible
changes`_ you'll want to be aware of when upgrading from tsr 0.3.5 or older
versions.

.. _`new features`: `What's new in tsr 0.3.6`_

What's new in tsr 0.3.6
=======================

Application state control
-------------------------

    * Add new functionality to the API and provisoners: stop and starting an
      App

Services
--------

    * Add support for plans in services

Backwards incompatible changes
==============================

tsr 0.3.6 does not introduce any incompatible changes.
